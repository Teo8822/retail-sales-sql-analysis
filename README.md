# Retail Sales SQL Analysis

End-to-end retail sales data analysis project using SQL and Python.

## Project Objectives
- Calculate key business KPIs (Revenue, AOV, Retention)
- Perform cohort analysis
- Analyze customer behavior
- Create business-ready insights

## Tech Stack
- SQL (joins, aggregations, window functions)
- Python (pandas, matplotlib)
- Data Cleaning & Transformation
- Exploratory Data Analysis (EDA)

## Key Business Insights
- Identified top revenue-generating segments
- Measured repeat purchase rate
- Calculated cohort retention trends
- Built KPI summary tables for reporting

## Skills Demonstrated
- Data modeling
- Analytical thinking
- Business metric calculation
- SQL querying for real-world datasets
